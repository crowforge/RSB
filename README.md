# RSB (R Service Bus)

### Build

Pre-requisite:

- Sun/Oracle JDK 1.6
- Maven 3

To produce rsb.war, run:

    mvn clean install

### Integration tests

Run:

    mvn -Pit verify


### Deployment

RSB should deploy fine on any Java web-container supporting Servlet 2.5.
